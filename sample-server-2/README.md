# GrapghQL Server

- GraphQL Server using NodeJS and JSON-Server

## Use

- Install Dependencies

```bash
$ npm install
```

- Run JSON-Server (Port 3000)

```bash
$ npm run json:server
```

- Run Server (Port 4000)

```bash
$ npm run dev:server
```

- Visit Graphiql IDE

Go to http://localhost:4000/graphql
